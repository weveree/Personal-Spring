# Personal-Spring
Personal-Spring is a project aimed at building a tool for creating fast and efficient APIs. Its goal is to replicate the development approach and syntax of Spring Boot, while remaining lightweight and easy to use.
## Start
You need to create a `.env` file in your project's root.


```.env
SERVER.PORT=9999
DATABASE.HOST=host
DATABASE.PORT=5432
DATABASE.NAME=name
DATABASE.USER=user
DATABASE.PASSWORD=password
```
