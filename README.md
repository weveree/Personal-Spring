# Personal-Spring
Personal-Spring is a project that tries to build a tool that lets us create fast APIs. It is also trying to replicate Spring-Boot's way of working and synthax.
## Start
You need to create a `.env` file in your project's root.


```.env
SERVER.PORT=9999
DATABASE.HOST=host
DATABASE.PORT=5432
DATABASE.NAME=name
DATABASE.USER=user
DATABASE.PASSWORD=password
```
